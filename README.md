# Система учета счет-фактур

Простая и удобная система для учета счетов, клиентов и платежей.

## Установка

1. Установите XAMPP с PHP 7.4+
2. Скопируйте все файлы в папку `htdocs/accounting-system/`
3. Откройте в браузере `http://localhost/accounting-system/install.php`
4. База данных будет создана автоматически
5. Войдите в систему с логином `admin` и паролем `admin123`

## Возможности

### Для кассира:
- Просмотр всех счетов
- Внесение оплат (полностью или частично)
- Просмотр клиентов
- Создание новых счетов
- Просмотр отчетов

### Для администратора (все возможности кассира +):
- Удаление/редактирование счетов
- Управление клиентами (добавление/удаление)
- Настройки системы

## Структура базы данных

1. `clients` - клиенты
2. `invoices` - счета-фактуры
3. `payments` - платежи
4. `users` - пользователи системы

## Безопасность

- Все пароли хранятся в хэшированном виде
- Защита от SQL-инъекций через подготовленные выражения
- XSS защита через htmlspecialchars()
- Проверка авторизации на каждой странице
- Разделение ролей (кассир/администратор)

## Дополнительные возможности

- Поиск по всем полям
- Фильтрация по статусам и датам
- Экспорт отчетов в Excel/PDF
- Печать отчетов
- Адаптивный дизайн для мобильных устройств
- Минималистичный и современный интерфейс

## Обновление

Для обновления просто замените файлы в папке `htdocs/accounting-system/`
(кроме файла `config.php`, если вы меняли настройки подключения к БД)

## Поддержка

При возникновении вопросов:
1. Проверьте логи ошибок Apache (`logs/error.log`)
2. Убедитесь что PHP расширения pdo_mysql включены
3. Проверьте права доступа к папкам

## Лицензия

Система распространяется бесплатно для личного использования.
